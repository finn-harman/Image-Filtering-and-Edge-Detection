# Image-filtering-and-Edge-Detection

An exploration of image filtering and edge detection in Python.
* Moving average filter
* Sobel filter
* 2D Gaussian filter
* Laplacian filter
* 2D convolution filter (Conv2D)

## Built With

* [NumPy]
* [Matplotlib]
* [SciPy]
* [PyTorch]
